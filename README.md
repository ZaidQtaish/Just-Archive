## Getting Started (Development)

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
git clone https://github.com/rknastenka/Just-Archive.git
cd Just-Archive
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
